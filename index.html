Project Idea

Overview

For my project idea I am planning on creating a video game, I want the game to be multiplayer where there is some type of interaction between players. Within the game I want there to be minimal lag and delay so that the user experience is good. Regarding the type of game, I am aiming for either a strategic / cooperation type of game or one where the players will verse each other, this is still unclear and will need further decision making before coming to a final point.

Motivation

Firstly, creating a video game would help me build my portfolio/resume, as working in the game industry is a potential future I have for myself. By creating my own game, I will be able to develop new skills and have some experience which will benefit me in the future. Secondly, I myself find great enjoyment in playing video games and I am really passionate about them. So creating a video game of my own would be challenging but also enjoyable.

Description

As this will be the first game I ever make, I know there will be many limitations regarding how far I can get within the development stages. However, I already have the plan of making it some type of multiplayer video game where two to four players can at least somewhat interact with each other. I am going to be making my video game a software which people run on their computers and not one that they play on an internet browser as I think this provides a much better experience for the user. The player will be able to control their character's movement and interaction with the world using a set of keybinds on a keyboard. The final goal of this game will be to survive till the end by cooperating with each other and defeating enemies that spawn at an increasing rate. I want to include a random drop rate system where players get drops based off a RNG system when they kill the enemies. As this will be a survival type of game I want the terrain / map to be different each time it is played, to do so I need to use procedural map generation to ensure that the game can be played more than once and that players won't get bored. The game will need to be able to generate random worlds with different seeds. I want the game to be survival / roguelike meaning the player progresses and gathers items to get stronger and finally complete the game, but if they die they restart on a new world with nothing from the previous game session. I'm also planning on having there be multiple difficulties so that those that want a challenge can have one. However there will be a drawback regarding my game, as I want it to be multiplayer, the players will need to do port forwarding to play with each other, but there may be an opportunity to make it easily playable once I do more research. Regarding the graphics of the video game I will most likely be creating the 3d models myself as I want it to be original.

Tools and Technologies

To create my game I will be using unity, as unity is a C++ based game engine I will need to learn how to code in C++. Additonally as I plan on designing the models in my game myself I can do that in unity itself which will be easier for me as I won't have to use another software and worry about whether I can use both of them in conjuction.

Skills Required

The main skills that are required for me to complete this project would be knowledge in programming with C++, thankfully there are many useful resources online and tutorials which I can use and learn fro, which will assist me in my project. Furthermore, unity is considered fairly beginner friendly so that will be extremely helpful when creating my own game.

Outcome

Finally, for my project to be a success, it would need to be a playable game. If I am able to achieve all my goals and what I am aiming for the game should look like a multiplayer survival / roguelike game where players need to survive waves of enemies until they complete the final stage. There will be multiple RNG elements such as the procedural map generation and the random drop system in which enemies will give random loot to help aid the player in their mission as they aim to complete the game.
